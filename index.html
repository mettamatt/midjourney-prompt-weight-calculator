Midjourney Multi Prompt Weight Calculator

This tool aids in generating accurately weighted multi prompts for Midjourney, a generative AI service creating images from natural language descriptions, known as prompts. Note: Additional parameters (if required) will have to be added by the user.

An example of how this tool works: If you want an image of the Swiss Alps but are getting unwanted elements like houses and trees, you could set the positive weight to 60% and the negative weight to 40%. Input "beautiful swiss alps" and "landscape" as positive prompts and "house" and "trees" as negative prompts. The script would generate the multi prompt: "beautiful swiss alps::30 landscape::30 houses::-20 trees::-20".

Follow the steps below to use this tool:

Detailed Instructions

  1. Global Weights: Set the positive weight using the "Positive Weight" input. The negative weight is automatically calculated as 100 minus the positive weight. This will help in guiding the Midjourney AI in focusing on specific elements of your preference.
  2. Generated Multi Prompt: The generated multi prompt will be displayed here. This multi prompt is the weighted combination of your input that you can directly use on Midjourney. The "Copy" button lets you easily copy the generated prompt.
  3. Positive Prompts: Specify the number of positive prompts in the "Number of Positive Prompts" input. Positive prompts guide the AI towards desired elements in the generated image.
  4. Negative Prompts: Specify the number of negative prompts in the "Number of Negative Prompts" input. Negative prompts guide the AI away from undesired elements.
  5. Clear Local Storage: To reset the tool and clear saved data, click the "Clear Local Storage" button.

For further details on using Midjourney and prompts, refer to the official Midjourney documentation.

Input the positive weight as a percentage. The tool will automatically calculate the negative weight as the remaining percentage.
Example: For an image of the Swiss Alps with less houses and trees, you could set the positive weight to 60% and the remaining 40% is considered as negative weight.

Example: For positive prompts like "beautiful swiss alps" and "landscape", and negative prompts like "house" and "trees", the tool will generate a corresponding multi prompt.

Positive Prompts

Indicate the number of positive prompts using the "Number of Positive Prompts" input. Then, provide the text for each positive prompt in their respective input boxes. These prompts direct the AI towards elements you want to see in the image.

Negative Prompts

Indicate the number of negative prompts using the "Number of Negative Prompts" input. Then, provide the text for each negative prompt in their respective input boxes. These prompts help the AI avoid elements you don't want in the image.
